## Link

```sh
ln -s ~/.config/tmux/.tmux.conf ~/.tmux.conf
```

## Hotkeys

`Control+A` - command mode
`Control+A + S` - show all sessions
`Control+A + D` - deatach
`Control+A + Shift + i` - isntall plugins
